# Hướng dẫn lập trình game trên AK Embedded Base Kit

Chào mừng các bạn Intern đến với dự án phát triển Game trên nền tảng STM32L151. Repository này cung cấp bộ khung (Base Kit) chuẩn giúp các bạn tập trung 100% vào việc sáng tạo logic game mà không cần lo lắng về cấu hình phần cứng.

## I. Giới thiệu

Để bắt đầu dự án cá nhân của mình, các bạn thực hiện đúng các bước sau:

- Đầu tiên các bạn truy cấp vào đường link sau: https://github.com/the-ak-foundation/ak-base-kit-stm32l151

- Fork: Nhấn nút Fork ở góc trên bên phải của repo này để copy dự án về tài khoản cá nhân của bạn.
